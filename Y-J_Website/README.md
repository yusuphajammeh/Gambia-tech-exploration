# Yusupha Jammeh - Portfolio Website

A simple, responsive portfolio website showcasing my journey as a student developer exploring technology in Gambia.

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Smooth Scrolling Navigation**: Easy navigation between sections
- **Contact Form**: Basic form for visitor messages (demo functionality)
- **Project Showcase**: Display of current projects and GitHub links

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. For local development, you can use a simple HTTP server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`
